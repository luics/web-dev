开发工具

浏览器

随着微软转向 Chromium 开源项目，市场中占绝大部分流量都在 Chrome 内核上，兼容性问题比起十多年前(2005年)已有了根本性的变化。本课程所有代码均可在 Chrome 浏览器中运行。

IDE

市场上适合 Web 开发的优秀 IDE 越来越多，这里仅罗列数个笔者使用过的 IDE：

  1. VSCode，免费，开源，性能卓越，功能丰富，推荐。
  2. Atom，免费，开源，性能较好，功能较丰富。
  3. Sublime Text 2，免费，性能卓越，功能配置较复杂。
  4. WebStorm，付费，性能卓越，功能丰富。

node

node 全称是 nodejs，原作者给 Chrome 浏览器的 V8 JS 引擎编写了 OS 桥阶层，让 node JS 代码具备了文件 IO、网络 IO、进程管理 等 OS 资源调度能力，从而极大扩展了 JS 的应用范围。

Node 能做什么?

  1. 开发工具
  2. 开发服务端
  3. 写应用程序（如 ChromeOS 中）
  4. 写 OS

在本地运行（请确保已下载了课程资源）：

node examples/web-dev/multiply.js

NPM

NPM 全称是 Node Package Manager，是 node 包管理器，可类比 Maven（Java）、Cocoapods(iOS)、GEM(Ruby)、PIP(Python)。我们会接触到 npm 命令和 npm 站点。
