# 开发环境

## 安装开发环境

1. 安装 [Chrome](https://www.google.cn/intl/zh-CN/chrome/)。
1. 安装 [VSCode](https://code.visualstudio.com/)（或任何 IDE）。
1. 安装 [node](https://nodejs.org)。 
1. 安装 HTTP 服务器：
    ```bash
    npm install -g cnpm --registry=https://registry.npm.taobao.org
    cnpm install -g serve
    serve ./
    ```
