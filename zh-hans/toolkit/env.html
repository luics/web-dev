开发环境

开发环境

安装开发环境

  1. 安装 Chrome。
  2. 安装 VSCode（或任何 IDE）。
  3. 安装 node。
  4. 安装 HTTP 服务器：
     npm install -g cnpm --registry=https://registry.npm.taobao.org
 cnpm install -g serve
 serve ./
  5. 安装课程资源。

练习 - 本地的页面

请在本地新建一个页面：

  1. 页面包含的 html、js 如下所示。
  2. 打开命令行，切换该文件所在目录，运行serve .。
  3. 浏览器中打开该页面http://localhost:5000/xxx.html。
  4. 打开 DevTools，熟悉 Elements、Console。
  5. DevTools 切换到 Sources，设置断点并单步执行。

See the Pen env-1 by luics (@luics) on CodePen.

练习 - Polyfill

写一个 Polyfill 实现 String.prototype.startsWith 的功能。

  1. 通过 MDN 了解 startsWith。
  2. 试着写一个类似的 startsWith 方法，尽可能覆盖各种边界场景。
  3. 写一单元测试用例，可不依赖测试框架，也可以尝试 mocha 等框架。
